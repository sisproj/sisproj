5주차

※ 임시 저장함, 결재 대기함, 결재 완료함, 결재 반려함 : 각리스트 구조는 비슷하니 한번만 설명 // 임시저장함(리스트).png
 ＃ 공통 설명
  - 아직 읽지 않은 글은 제목이 bold체면서 N이미지 추가됨
  - 검색 기능 : 문서번호나 제목, 내용으로 검색 됨
  - 페이지 기능 : 한페이지에 15개 레코드 출력, 7페이지씩 한 블럭으로 구성

※ 리스트별 설명
  - 임시 저장함 : 본인이 임시로 저장한 본인의 문서들만 보여줌
  - 결재 대기함 : 본인의 결재 대기중인 문서들과 다른사람이 작성하였지만 자기차례의 결재 문서도 보여줌
  - 결재 완료함 : 본인의 결재 완료된 문서들만 보여줌
  - 결재 반려함 : 본인의 결재 반려된 문서들만 보여줌

※ 상세보기 (임시저장 문서) //임시저장함(상세).png
  - 본인이 임시로 저장해둔 문서를 볼 수 있다.
  - 임시로 저장된 문서는 수정, 삭제가 가능하다.

※ 상세보기 (결재대기) //결재대기(상세).png
  - 본인의 문서라면 결재 진행 상태를(결재자란에 서명 되는것 확인) 볼 수 있다.(결재대기)
  - 타인의 문서고 내가 결재해야 된다면 상단에 승인, 반려 버튼이 나온다.(결재대기)
  - 현재 문서 상태가 결재 대기로 표시됨
  - 결재자가 몇일에 결재 했는지 결재 라인에 표시됨
  - 의견을 첨언할 수 있음 (의견은 임시저장함에서는 뜨지 않음. 결재대기일때만 수정 등록 삭제 가능, 결재 반려 혹은 완료라면 이미 등록된 것만 볼 수 있고 수정 등록 삭제는 불가)

※ 결재 환경 설정 
 ＃ 결재서명등록 // 결재환경설정(서명등록).png
  - 결재에 사용될 본인의 사인을 이미지로 등록
  - 업로드창이 이미지 파일만 보여주도록 디폴트 // (서명등록-업로드창).png
  - 모든 파일 보기로 바꿔서 다른 파일을 올리더라도 이미지만 올라갈 수 있도록 (유효성) 검사 해둠 결재환경설정 // 결재환경설정(서명등록-유효성).png
  - 다른 이미지 등록시 기존 이미지 삭제 후 새 이미지로 등록 

 ＃결재라인등록 : 신규등록 // 결재환경설정(라인등록).png
  - 신규등록 버튼 클릭시 왼쪽 조직도가 올라옴, 라인 제목부분도 신규 결재라인 등록으로 변경됨 // 결재환경설정(라인등록-신규등록).png
  - 조직도에서 더블클릭하면 결재라인 리스트에 추가됨 
   1. 본인은 직접 추가할수 없으며 본인 이외의 사람을 더블클릭하면 1번으로 자동 추가됨
   2. 추가된 리스트에서 다시 더블클릭하면 제거됨
   3. 이미 추가된 사람은 더이상 추가되지 않음
  
  - 라인이 완성되고 저장버튼 클릭시 제목 입력창이 나타남 // 결재환경설정(라인등록 - 라인저장).png
   1. 모달같은 설정으로 다른 곳 클릭 안되게 처리함
   2. 제목 입력후 저장 누르면 왼쪽 결재라인에 추가됨

 ＃결재라인등록 : 상세보기 // 결재환경설정(라인상세).png
  - 왼쪽 결재라인 리스트를 클릭하면 오른쪽에 리스트로 보여줌
  - 삭제를 원하면 삭제 버튼 클릭
  - 저장된 리스트에서 더블클릭하면 리스트에서 삭제되고 조직도에서 더블클릭하면 새로 추가되고 제목부분에 수정중... 이 붙음 // 결재환경설정(라인상세-수정중).png
  - 수정 버튼 클릭시 신규 등록처럼 제목 입력 부분이 뜸. 그리고 저장 누르면 저장됨. // 결재환경설정(라인상세-수정제목입력).png

 ＃결재라인등록 : 저장된 결재라인 삭제 // 결재환경설정(라인상세-삭제).png
  - 한번더 물어봐주고 삭제됨 // 결재환경설정(라인상세-삭제 후).png
